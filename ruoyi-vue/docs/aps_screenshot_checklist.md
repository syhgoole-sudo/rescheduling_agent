# APS 项目截图清单

## 1. 系统首页与菜单

- 截图名称：`01_aps_menu.png`
- 页面路径：若依首页 / APS 调度系统菜单
- 截图用途：证明 APS 模块已集成到若依菜单中。

## 2. 产品管理

- 截图名称：`02_product_management.png`
- 页面路径：APS 调度系统 / 产品管理
- 截图用途：展示基础数据管理能力。

## 3. 订单管理

- 截图名称：`03_order_management.png`
- 页面路径：APS 调度系统 / 订单管理
- 截图用途：展示 NORMAL 普通订单和 INSERT 插单订单。

## 4. 调度方案列表

- 截图名称：`04_schedule_plan_list.png`
- 页面路径：APS 调度系统 / 调度方案
- 截图用途：展示 INITIAL 和 RESCHEDULE 方案，以及方案状态。

## 5. 调度任务明细

- 截图名称：`05_schedule_task_by_plan.png`
- 页面路径：调度方案 / 查看任务
- 截图用途：展示某个方案下的任务明细、设备、时间和工序。

## 6. 单方案甘特图

- 截图名称：`06_single_gantt.png`
- 页面路径：调度方案 / 查看甘特图
- 截图用途：展示按设备维度的任务排产结果。

## 7. 插单事件列表

- 截图名称：`07_insert_event_list.png`
- 页面路径：APS 调度系统 / 插单事件
- 截图用途：展示插单事件、原方案、新方案、事件状态和策略类型。

## 8. 影响分析结果

- 截图名称：`08_impact_analysis.png`
- 页面路径：插单事件 / 查看分析
- 截图用途：展示影响等级、受影响任务数、受影响订单数、受影响设备数。

## 9. 策略推荐结果

- 截图名称：`09_strategy_recommendation.png`
- 页面路径：插单事件 / 推荐策略
- 截图用途：展示策略类型、策略名称、冻结任务数、可调整任务数和插单任务数。

## 10. 局部重调度生成结果

- 截图名称：`10_local_reschedule_result.png`
- 页面路径：插单事件 / 生成重调度
- 截图用途：展示新方案 ID、原方案 ID、任务数量和 KPI 摘要。

## 11. KPI 对比

- 截图名称：`11_kpi_compare.png`
- 页面路径：调度方案 / 方案对比
- 截图用途：展示新旧方案延期、makespan、变更任务、插单任务和冻结任务指标。

## 12. 新旧甘特图对比

- 截图名称：`12_gantt_compare.png`
- 页面路径：调度方案 / 甘特图对比
- 截图用途：展示 INITIAL 原方案与 RESCHEDULE 新方案的任务变化。

## 13. 解释报告

- 截图名称：`13_explanation_report.png`
- 页面路径：插单事件 / 解释报告
- 截图用途：展示插单概述、影响说明、策略说明、KPI 说明、收益、代价和调度建议。

## 14. 采用方案

- 截图名称：`14_confirm_plan.png`
- 页面路径：调度方案 / 采用方案
- 截图用途：展示调度员确认采用 RESCHEDULE 方案后的状态流转。

## 15. 拒绝方案

- 截图名称：`15_reject_plan.png`
- 页面路径：调度方案 / 拒绝方案
- 截图用途：展示调度员拒绝 RESCHEDULE 方案并恢复原方案的状态流转。

## 16. 推荐截图组合

用于简历或面试展示时，建议优先保留以下 6 张：

- `04_schedule_plan_list.png`
- `06_single_gantt.png`
- `08_impact_analysis.png`
- `11_kpi_compare.png`
- `12_gantt_compare.png`
- `13_explanation_report.png`
